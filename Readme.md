# Common Utils

It is a NPM package usefull for common utilities.

## Installation


If you are using npm 

```bash
npm i @deepakthapa/cmutils
```
or yarn
```bash
yarn add @deepakthapa/cmutils
```

## Usage

```javascript
import {isNumeric} from "@deepakthapa/cmutils"
if(isNumeric(12)){
    // Do something
}
```
